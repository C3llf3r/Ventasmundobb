# Sistema de Ventas e Impuestos (CLP)

Esta aplicación permite registrar ventas diarias y calcular el IVA automáticamente.

## Cómo iniciar la aplicación

1.  Abrir una terminal en esta carpeta.
2.  Ejecutar: `npm run dev`
3.  Abrir el navegador en la dirección que aparece (usualmente `http://localhost:5173`).

## Características

*   **Registro Diario**: Ingresa efectivo y tarjeta.
*   **Dashboard**: Ve totales y cálculo de impuestos (19%).
*   **Historial**: Revisa y borra ventas del mes.
*   **Datos Locales**: Por ahora, los datos se guardan en tu navegador.

## Nube (Firebase)

Para habilitar el respaldo en la nube, necesitamos configurar Firebase. Ejecuta el comando de login si deseas activarlo.
